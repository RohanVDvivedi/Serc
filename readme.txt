This project is for a c server side framework for rest api.

src for .c c code files
inc for .h header files
obj for .o object files
bin for .out project run file
con for .json configuration file
	relating to database, and other libraries included
shl for .sh build scripts that you may require
pyt for .py python helper scripts
log for .lg log text files

-> It is a frame work that helps host rest api service, and static files over HTTP/TCP/IP protocal
